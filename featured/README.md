# CCNA Network Labs (Featured Projects)

This repository contains a collection of structured networking labs based on CCNA-level concepts. The labs are organized by major networking domains to demonstrate practical implementation of core technologies commonly used in enterprise environments.

Each folder represents a specific networking domain and includes configuration, topology, and validation steps.

---

## 📁 Project Structure

- **vlan-intervlan/**  
  VLAN creation, trunking, and inter-VLAN routing

- **stp-etherchannel/**  
  Layer 2 loop prevention using STP and link aggregation using EtherChannel

- **ospf-routing/**  
  Dynamic routing using OSPF in a single-area topology

- **acl/**  
  Traffic filtering using standard and extended Access Control Lists

- **dhcp-nat/**  
  DHCP service configuration along with static and dynamic NAT for internet access

- **network-redundancy/**  
  Gateway redundancy using HSRP

- **network-security/**  
  NTP server/client and secure device management (SSH) configuration along with Layer 2 security features including Port Security, DHCP Snooping, and Dynamic ARP Inspection

---

## 🎯 Purpose of These Labs

The goal of these labs is to demonstrate:

- Practical understanding of enterprise network design concepts
- Ability to configure and troubleshoot common networking protocols
- Familiarity with Cisco IOS configuration in realistic topologies

---

## 🧠 Key Skills Demonstrated

- VLAN segmentation and inter-VLAN routing
- Spanning Tree Protocol (STP) and loop prevention
- EtherChannel (LACP) configuration
- OSPF routing and adjacency formation
- ACL-based traffic control
- DHCP services and NAT configuration
- High availability using HSRP
- Layer 2 security mechanisms
- Secure device management (SSH)

---

## 📌 Notes

- These labs are based on structured exercises completed during CCNA-level study.
- Each folder contains its own topology, configuration files, and validation steps where applicable.
- The focus is on understanding, implementation, and reproducibility of each concept.
