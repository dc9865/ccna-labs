# Other CCNA Labs

This folder contains a collection of **supplementary labs** covering a wide range of CCNA topics.

These labs are designed to reinforce fundamental concepts and provide hands-on exposure to various networking features. While not as in-depth as the featured labs, they demonstrate broad familiarity with real-world configurations and protocols.

---

## 📚 Topics Covered

* **Basic Configuration** – Initial device setup and management access
* **CDP / LLDP** – Neighbor discovery and device identification
* **DNS** – Name resolution concepts and configuration
* **DHCP / VTP** – IP address management and VLAN propagation
* **FTP / TFTP** – File transfer and configuration backup methods
* **GRE Tunnel** – Basic tunnel setup and verification
* **IPv4 / IPv6** – Addressing and dual-stack configuration
* **SNMP** – Network monitoring and management basics
* **SSH** – Secure remote access configuration
* **Static & Dynamic Routing** – Route configuration and protocol behavior
* **Syslog** – Logging and monitoring network events
* **VLSM (Subnetting)** – Efficient IP address allocation
* **Voice VLAN** – Segmentation for voice traffic
* **Wireless LAN** – Basic wireless configuration concepts
* **QoS** – Traffic prioritization and classification fundamentals

---

## 🎯 Purpose

* Reinforce core networking concepts through hands-on practice
* Build familiarity with a wide range of CCNA-level topics
* Support deeper understanding of configurations used in real environments

---

## 🛠️ Notes

Each subfolder contains:

* Device configurations
* Lab topology (if applicable)
* Supporting files for setup and verification

---

## 🚀 How to Use

You can explore each folder individually based on the topic you want to review.
These labs are especially useful for:

* Quick refreshers before interviews
* Strengthening weak areas
* Practicing configuration and verification commands

---

## ⚠️ Disclaimer

These labs focus on **fundamentals and concept validation**, not production-level design.
